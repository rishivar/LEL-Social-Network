# LEL-Social-Network

High School Project

ABOUT THE PROJECT

LEL is a personalized entertainment site that aims in engaging the user with games, quiz, news and several other visually appealing real-world browser emulation. 

The program effectively and coherently implements Object-oriented programming, Data file handling and several other intricate components of C++ to bring out a perfect simulation. With an ultimate aim to fulfil the expectations of the user, LEL has everything to offer.
